Relation changes are indicated with the same color code as in game.

A lock icon means the option or event is unlocked by a previous action or dialogue choice. An hourglass icon means the event occurs if you take too much time. A skull icon means the option or event is unlocked by the death of a character. A tv icon indicates dialogues heard on televisions or screens.
